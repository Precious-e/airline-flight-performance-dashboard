# Airline Flight Performance Dashboard

## Overview

This Power BI project analyzes over 1.9 million airline flight records to evaluate operational performance, delays, cancellations, and airline efficiency.

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling

## Key Metrics

- Total Flights: 1.9M
- Delayed Flights: 790.5K
- Cancelled Flights: 28.6K
- On-Time Rate: 58%
- Delay Rate: 40.5%
- Cancellation Rate: 1.5%

## Dashboard Features

- Flight volume by city
- Delay analysis by airline
- Cancellation reasons
- Weekly cancellation trends
- Flight status monitoring

## Key Insights

- Atlanta handled the highest flight traffic.
- Over 40% of flights experienced delays.
- Weather was a leading cause of cancellations.
- Airline performance varied significantly.
